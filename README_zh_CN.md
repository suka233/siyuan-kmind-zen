# KMind Zen 思源插件（公测）

[English](https://github.com/suka233/siyuan-kmind-zen/blob/main/README.md)

KMind Zen 是全新的专业导图工具，起步于思源，走向更广阔的世界，官网 https://kmind.app

与上版本 KMind2 不同，KMind Zen 完全从 0 开始，重新设计了内核、UI、交互等，以期望获得更高的自主性和拓展性。当前 KMind Zen 提供思源插件、Obsidian 插件、网页应用、[KMind Zen OpenClaw Skill](https://clawhub.ai/suka233/kmind-markdown-to-mindmap)（Skill 是纯离线且通用的，你可以在任意支持使用 Skill 的地方调用，可以获得直接从文本转为导图图片的能力），独立桌面端也即将到来。

## 0.2.0 更新（2026-04-29）

- 将共享 KMind 内核升级到 0.2.0。
- 修复极端情况下的化学方程式渲染问题，包括 mhchem 条件标签中包含中文时的显示异常。
- 更新思源 Dock 栏图标视觉效果，改为更清爽的宿主单色字形，以更好适配思源浅色和深色主题。

## 特点

- 所有端的 KMind Zen 都由同一个内核驱动，意味着一份 KMind Zen 源文件可以在思源、Obsidian、网页端、桌面端无缝流转。你甚至可以使用任意 AI 助手，通过 KMind Zen Skill 直接根据传入的文档离线生成可以二次编辑的导图。一个经典使用场景：使用 OpenClaw 自动将课堂录音/会议录音转为 KMind Zen 源文档，便于二次编辑后存档。
- 精心设计过的源文档格式：`.kmindz.svg`，既包含所有源文件内容，又因为实际上是 SVG 图片格式，所以无需打开也能直接预览导图。
- 智能主题，所有官方主题都设计有 light/dark 两种模式，无需手动切换主题，可以自动切换明亮/暗黑模式。
- 无限概要节点，可以给概要再加上概要，无限层级。
- 优化过的节点挖空功能，节点和备注都可以任意挖空，用于辅助记忆。
- 优化过的公式编辑器，使用斜杠菜单快捷调出公式编辑器。
- 优化过的全局搜索，不仅可以搜节点内容，还能搜索备注、批注内容等。
- 节点双链功能。
- 节点批注功能，不同于备注，可以按时间展示节点上的批注，用于回顾整个思考过程。
- 优化过后的拖拽交互，尽量让拖拽节点的手感更上一层楼。
- 优化后的节点超链接功能，可以给一个节点添加多个超链接，并且每个超链接可以设置 icon。
- 优化后的禅模式、只读模式体验，可以方便地在右上角看到当前状态并退出当前模式。
- 尽力优化的移动端体验，为移动端提供了专属 UI，当然，还需要您的反馈才能更加完善。
- 其它基础功能（富文本节点、富文本备注、多根节点、节点图片、节点 TODO、节点 icon、节点备注、节点标签、格式刷、关联线）。

## 思源特供

- 直接在思源文档树中创建 KMind Zen 导图，天然支持思源自带的标签、超链接等。
- 直接拖拽思源文档树的文档进入导图，成为 `文档卡`；`文档卡` 是特殊节点，可以任意添加子节点、关联线等。文档卡默认为预览模式，预览模式下为了保障性能，会低频率更新预览内容，您可以随时切换为实时模式。
- 直接拖拽思源块进入导图，成为 `块卡`；`块卡` 是特殊节点，可以任意添加子节点、关联线等。
- 在思源 Dock 栏中创建导图。
- 得益于思源的块设计，导图和导图节点都可以复制出永久超链接。该超链接可以任意粘贴到外部软件，点击后会自动打开思源，然后打开该导图并跳转到指定节点。
