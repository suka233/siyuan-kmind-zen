# KMind Zen 思源插件（公测）

[English](https://github.com/suka233/siyuan-kmind-zen/blob/main/README.md)

KMind Zen 是全新的专业导图工具，起步于思源，走向更广阔的世界，官网 https://kmind.app

与上版本 KMind2 不同，KMind Zen 完全从 0 开始，重新设计了内核、UI、交互等，以期望获得更高的自主性和拓展性。当前 KMind Zen 提供思源插件、Obsidian 插件、网页应用、[KMind Zen OpenClaw Skill](https://clawhub.ai/suka233/kmind-markdown-to-mindmap)（Skill 是纯离线且通用的，你可以在任意支持使用 Skill 的地方调用，可以获得直接从文本转为导图图片的能力），独立桌面端也即将到来。

## 0.28.0 更新（2026-06-27）

- 新增节点右键菜单「添加父节点」与对应快捷键配置。
- 新增上移节点、下移节点功能，快捷键为 `Alt/Option+Up`、`Alt/Option+Down`。
- 新增收缩、展开节点快捷键：`Alt/Option+Left`、`Alt/Option+Right`。
- 更新内核版本。

## 特点

- 所有端的 KMind Zen 都由同一个内核驱动，意味着一份 KMind Zen 源文件可以在思源、Obsidian、网页端、桌面端无缝流转。你甚至可以使用任意 AI 助手，通过 KMind Zen Skill 直接根据传入的文档离线生成可以二次编辑的导图。一个经典使用场景：使用 OpenClaw 自动将课堂录音/会议录音转为 KMind Zen 源文档，便于二次编辑后存档。
- 精心设计过的源文档格式：`.kmindz.svg`，既包含所有源文件内容，又因为实际上是 SVG 图片格式，所以无需打开也能直接预览导图。
- 智能主题，所有官方主题都设计有 light/dark 两种模式，无需手动切换主题，可以自动切换明亮/暗黑模式。
- 本地主题设计器和主题库，支持通过 `.kmind-theme.json` 离线主题包导入 / 导出分享主题。
- 支持更稳定的 SVG / PNG 导出、PNG 图片美化导出和复制为图片样式选择，方便把导图整理成更适合分享的图片。
- 支持在画布中直接拖拽调整节点宽度。
- 支持真正的节点内原地编辑，编辑区保持在节点正文内部。
- 大纲与分屏模式，可以用空间导图和连续大纲两种方式编辑同一份导图。
- 分屏模式支持导图节点与大纲双向拖拽。
- 高级关联线编辑，支持直线 / 正交连线 / 圆角正交连线、编辑器智能避让、虚线 / 点线样式、颜色配置、项目级默认配置，以及概要关联线。
- 无限概要节点，可以给概要再加上概要，无限层级。
- 优化过的节点挖空功能，节点和备注都可以任意挖空，用于辅助记忆。
- 优化过的公式编辑器，使用斜杠菜单快捷调出公式编辑器。
- 优化过的全局搜索，不仅可以搜节点内容，还能搜索备注、批注内容等。
- 节点双链功能。
- 节点批注功能，不同于备注，可以按时间展示节点上的批注，用于回顾整个思考过程。
- 优化过后的拖拽交互，支持多选后批量拖拽节点，尽量让拖拽节点的手感更上一层楼。
- 右键菜单支持按层级展开和快速展开 / 收起，便于聚焦大型导图。
- 可配置画布拖拽习惯，适配偏好空白处平移或空白处框选的用户。
- 优化后的节点超链接功能，可以给一个节点添加多个超链接，并且每个超链接可以设置 icon。
- 优化后的禅模式、只读模式体验，可以方便地在右上角看到当前状态并退出当前模式。
- 尽力优化的移动端体验，为移动端提供了专属 UI，当然，还需要您的反馈才能更加完善。
- 可配置的 KMind Zen 快捷键，并且只在当前导图体验中生效，不影响普通思源编辑。
- 其它基础功能（富文本节点、富文本备注、多根节点、节点图片与图片快速调整、节点 TODO、节点 icon、节点备注、节点标签、格式刷、关联线）。

## 思源特供

- 直接在思源文档树中创建 KMind Zen 导图，天然支持思源自带的标签、超链接等。
- 普通思源文档可以一键切换为导图视图，支持完整块树只读预览、标题大纲整理，并可创建为当前文档的可编辑子文档导图。
- MOC 文档树导图可以把思源文档树显示为导图，并将新建、重命名、拖动调整和删除等整理操作写回思源文档树。
- 可以把 KMind Zen 导图或选中节点复制为思源镜像块，粘贴到普通正文后只读展示，并随源导图保存自动刷新。
- 可以从思源斜杠菜单插入 KMind Zen 块导图，正文显示为原生图片预览，并可通过图片右上角编辑按钮继续编辑；块导图预览图清晰度可在全局设置中调整。
- 可以从思源 PDF 阅读器复制标注，并粘贴到导图中作为可点击的 PDF 链接。
- 拖拽思源文档树的文档到导图空白处可创建 `文档卡`，拖到已有节点可快速添加思源文档链接；按住 `Alt/Option` 拖到节点时可创建文档卡子节点。`文档卡` 是特殊节点，可以任意添加子节点、关联线等。文档卡默认为预览模式，预览模式下为了保障性能，会低频率更新预览内容，您可以随时切换为实时模式。
- 拖拽思源文档树中的 KMind 导图文档到导图时会作为链接引用：拖到空白画布创建普通链接节点，拖到普通节点追加思源文档链接。
- 拖拽思源块到导图空白处可创建 `块卡`，拖到已有节点可快速添加思源块链接；按住 `Alt/Option` 拖到节点时可创建块卡子节点。`块卡` 是特殊节点，可以任意添加子节点、关联线等。
- 复制 `siyuan://blocks/<id>` 或 Markdown 形式的思源块链接后可直接粘贴到导图中。粘贴到空白画布会创建正文可读的链接节点；粘贴到空白正文节点会自动使用思源块文本，粘贴到已有正文节点只追加外链。
- 在思源 Dock 栏中创建导图。
- 得益于思源的块设计，导图和导图节点都可以复制出永久超链接。该超链接可以任意粘贴到外部软件，点击后会自动打开思源，然后打开该导图并跳转到指定节点。

## 限制

- KMind Zen 尚未包含 KMind2 的所有功能，主要包括：更丰富的导入导出功能和少量旧版高级工作流。上述功能会以 KMind Zen 的风格重新设计后逐一推出。
- 文档树导图、思源文档一键转导图、MOC 和新建镜像块需要 Pro 授权；已有镜像块会继续正常阅读和刷新；Dock 导图永久免费。
- 思源文档导图的完整块树模式是只读预览，复杂块创建为正式导图后进入编辑时，会转换为 KMind Zen 当前支持的可编辑形态，预览排版可能不完全保留。
- 如果您使用思源同步，请不要同时多端修改同一张导图
- 使用新版本保存过的项目级 / root 级间距配置或项目级关联线配置，如果再被旧版本 KMind Zen 打开并保存，旧版本可能无法保留这些新增设置。
